# 心理健康平台 - 用户操作指南

## 如何预约心理咨询师

### 前提条件

1. 已注册并登录系统
2. 后端服务正常运行
3. 数据库中有咨询师数据

---

## 完整预约流程

### 步骤 1：浏览咨询师列表

1. **登录系统**
   - 访问 `http://localhost:5173/login`
   - 输入用户名和密码
   - 点击"登录"按钮

2. **进入咨询师页面**
   - 方式一：点击左侧导航栏的"咨询师"菜单
   - 方式二：直接访问 `http://localhost:5173/counselor`

3. **浏览咨询师列表**
   - 查看咨询师卡片信息：
     - 头像
     - 姓名
     - 专长领域
     - 评分（星级）
     - 咨询价格

4. **使用筛选功能（可选）**
   - 按专长筛选：选择您需要的专长领域
   - 按评分排序：选择评分高低排序
   - 搜索功能：输入咨询师姓名搜索

---

### 步骤 2：查看咨询师详情

1. **点击咨询师卡片**
   - 点击任意咨询师卡片上的"查看详情"按钮
   - 或直接点击卡片区域

2. **查看详细信息**
   - **基本信息**：
     - 姓名、职称
     - 执业资格证书
     - 从业年限
   - **专长领域**：
     - 擅长的咨询方向
     - 适合的人群
   - **个人简介**：
     - 专业背景
     - 咨询理念
     - 工作经验
   - **咨询价格**：
     - 单次咨询费用
     - 咨询时长

3. **查看用户评价**
   - 滚动到页面下方
   - 查看其他用户的评价和评分
   - 了解咨询师的服务质量

---

### 步骤 3：发起预约

1. **点击"预约咨询"按钮**
   - 位于咨询师详情页面顶部
   - 点击后弹出预约对话框

2. **填写预约信息**

   **a. 选择日期**
   - 点击日期选择器
   - 选择您希望咨询的日期
   - 注意：过去的日期会被禁用

   **b. 选择时间段**
   - 系统会显示该日期的可用时段
   - 绿色表示可预约
   - 灰色表示已被预约
   - 点击选择您方便的时间段

   **c. 选择咨询方式**
   - 在线咨询：通过视频/语音进行
   - 线下咨询：到咨询室面对面咨询

   **d. 填写备注（可选）**
   - 简要说明您的咨询需求
   - 特殊要求或注意事项

3. **确认预约**
   - 检查预约信息是否正确
   - 点击"确认预约"按钮
   - 等待系统处理

4. **预约成功**
   - 系统显示"预约成功"提示
   - 预约信息会发送到您的通知中心
   - 咨询师会收到预约通知

---

### 步骤 4：管理预约

1. **查看预约记录**
   - 点击顶部导航栏的"个人中心"
   - 选择"我的预约"
   - 或直接访问 `http://localhost:5173/appointments`

2. **预约列表**
   - 查看所有预约记录
   - 预约状态说明：
     - 🟡 **待确认**：咨询师尚未确认
     - 🟢 **已确认**：咨询师已确认，等待咨询
     - 🔵 **已完成**：咨询已完成
     - 🔴 **已取消**：预约已取消

3. **筛选预约**
   - 点击状态标签页
   - 查看不同状态的预约

4. **取消预约**
   - 找到需要取消的预约
   - 点击"取消预约"按钮
   - 确认取消操作
   - 注意：已确认的预约取消可能需要提前通知

5. **查看预约详情**
   - 点击"查看详情"按钮
   - 查看完整的预约信息
   - 包括咨询师联系方式（如有）

## 常见问题

### Q1: 如何知道预约是否成功？

**A:** 预约成功后，您会收到：

1. 页面上的成功提示
2. 通知中心的预约通知
3. 预约记录会出现在"我的预约"列表中

### Q2: 预约后多久会被确认？

**A:** 通常咨询师会在 24 小时内确认预约。您可以在"我的预约"中查看状态。

### Q3: 如何取消预约？

**A:**

1. 进入"我的预约"页面
2. 找到要取消的预约
3. 点击"取消预约"按钮
4. 确认取消操作

### Q4: 取消预约有时间限制吗？

**A:** 建议至少提前 24 小时取消预约，以便咨询师调整时间安排。

### Q5: 可以修改预约时间吗？

**A:** 目前需要先取消原预约，然后重新预约新的时间。

### Q6: 如何联系咨询师？

**A:** 预约确认后，您可以在预约详情中查看咨询师的联系方式。

### Q7: 在线咨询如何进行？

**A:** 预约确认后，系统会提供在线咨询的链接或房间号。

### Q8: 咨询费用如何支付？

**A:** 目前支付功能正在开发中，具体支付方式请咨询客服。

### Q9: 可以预约多个咨询师吗？

**A:** 可以，您可以同时预约多个咨询师，但请注意时间不要冲突。

### Q10: 预约记录会保存多久？

**A:** 所有预约记录会永久保存，您可以随时查看历史预约。

## 附录：API 接口说明（开发者参考）

### 预约相关接口

#### 1. 获取咨询师列表

```
GET /api/counselors
参数：
  - page: 页码
  - size: 每页数量
  - specialty: 专长筛选
  - sort: 排序方式
```

#### 2. 获取咨询师详情

```
GET /api/counselors/{id}
参数：
  - id: 咨询师ID
```

#### 3. 获取可用时段

```
GET /api/counselors/{id}/available-slots
参数：
  - id: 咨询师ID
  - date: 日期
```

#### 4. 创建预约

```
POST /api/appointments
请求体：
{
  "counselorId": 咨询师ID,
  "appointmentDate": "预约日期",
  "timeSlot": "时间段",
  "consultationType": "咨询方式",
  "notes": "备注"
}
```

#### 5. 获取预约列表

```
GET /api/appointments
参数：
  - status: 状态筛选
  - page: 页码
  - size: 每页数量
```

#### 6. 取消预约

```
PUT /api/appointments/{id}/cancel
参数：
  - id: 预约ID
```
